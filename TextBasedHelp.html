Welcome to "Power in your pocket": Text Based Help!

Phones: Top Tips: Include a simple easy to remember pin to secure your phone incase you lose it.

Tablets: Top Tips: Some tablets allow you to enable "Find My Device" so that in the event you lose it, you can try and track it down.

Laptops: Top Tips: As the laptop is portable it is very important that a good password is used when logging in. That way if someone were to steal the device or you were to lose it, they cannot access your files.

Computers: Top Tips: Having a computer set up with a few easy to understand icons on the main page can be great way to remove the effort when browsing the internet.

Smart TV's: Top Tips: A lot of Smart Tv's allow you to log in to various services using your mobile devices. This can save a lot of time as you will not need to use the on screen keyboard on the smart tv which can be very slow and confusing.
